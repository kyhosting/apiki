Cara Kerja

Setup dalam 4 langkah

Platform ini adalah gateway — semua request diteruskan langsung ke iVAS tanpa modifikasi data.

01
Daftar akun KY-SHIRO
Isi username, email, nomor WA. Verifikasi OTP yang dikirim ke WhatsApp kamu.
02
Login ke akun iVAS kamu
Di dashboard → Login iVAS. Masukkan email & password iVAS-mu. WebSocket otomatis terhubung.
03
Ambil API Key
Copy API Key dari halaman API Key. Pakai di bot Telegram, script Python, atau app apapun.
04
Terima SMS real-time
SMS masuk otomatis muncul di dashboard dan di-push ke SSE endpoint kamu. Tidak perlu polling.
python — bot_sms.py
import requests, sseclient, json # 1. Konfigurasi API Key API = "ky-xxxxxxxxxxxxxxxx" BASE = "https://yourdomain.com" HDR = {"X-API-Key": API} # 2. Stream SMS real-time (SSE) r = requests.get( f"{BASE}/api/sms/live/stream", headers=HDR, stream=True ) for ev in sseclient.SSEClient(r): if not ev.data: continue sms = json.loads(ev.data) print(f"SMS: {sms['message']}") # 3. Ambil OTP dari nomor d = requests.get(f"{BASE}/api/sms/otp", headers=HDR, params={ "number": "6281234567890", "range": "INDONESIA", }).json() for m in d["messages"]: print("OTP:", m["otp_candidates"]) # 4. Tambah nomor dari range requests.post(f"{BASE}/api/numbers/add", headers=HDR, json={"range_name": "INDONESIA"})
API Reference

Semua Endpoint Tersedia

Proxy langsung ke iVAS. Butuh X-API-Key di header.

🔒 Koneksi iVAS
POST/api/ivas/loginLogin ke akun iVAS sendiri
GET/api/ivas/statusStatus koneksi iVAS + WebSocket
POST/api/ivas/logoutLogout dari iVAS
⚡ SMS Monitoring
GET/api/sms/liveSMS live dari cache WebSocket
GET/api/sms/live/streamSSE push stream real-time
GET/api/sms/receivedHistori SMS per range + nomor
GET/api/sms/otpOTP dari nomor (auto-extract)
POST/api/sms/live/clearClear cache SMS live

GET/api/numbersNomor di range tertentu
