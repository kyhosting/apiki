Fitur

Lengkap, Real, Bukan Demo

Setiap fitur langsung terhubung ke iVAS. Tidak ada simulasi, tidak ada data palsu.

⚡
SMS Live Real-time
Terima SMS masuk secara instan lewat WebSocket iVAS. Dipush ke dashboard dan bisa di-stream via SSE.
✉
SMS Received
Ambil histori SMS yang sudah masuk per range dan nomor. Filter tanggal, range, SID platform.
☎
Ambil Nomor
Lihat Test Numbers dan My Numbers di akun iVAS kamu. Filter, search, ambil per range.
◈
Add / Delete Number
Tambah nomor dari Test Numbers ke My Numbers, atau kembalikan nomor. Bisa per ID atau per range.
◎
Cek Nomor
Verifikasi apakah nomor terdaftar, info range, A2P, dan termination ID-nya.
⚿
API Key
Pakai API Key untuk integrasi bot Telegram, Python script, Node.js, atau apapun.
📲
OTP via WhatsApp
Verifikasi dan reset password pakai OTP yang dikirim ke WhatsApp via Baileys (self-hosted).
⚙
Session Per User
Setiap user login ke iVAS dengan akun sendiri. Session terisolasi, aman, dan bisa reconnect otomatis.
📡
SSE Stream
Subscribe ke endpoint /api/sms/live/stream dan terima SMS otomatis dipush tanpa polling.
Integrasi

Mudah dipakai di mana saja

Satu header API Key, akses semua endpoint langsung dari iVAS.

Python — Terima SMS live
import requests, sseclient API = "ky-xxxxxxxxxxxx" BASE = "https://yourdomain.com" # Stream SMS real-time (SSE) r = requests.get( f"{BASE}/api/sms/live/stream", headers={"X-API-Key": API}, stream=True ) for ev in sseclient.SSEClient(r): if ev.data: sms = json.loads(ev.data) print(sms["originator"], sms["message"])
JavaScript — Bot Telegram
const API = "ky-xxxxxxxxxxxx"; const BASE = "https://yourdomain.com"; // Add nomor dari range const r = await fetch( `${BASE}/api/numbers/add`, { method: "POST", headers: {"X-API-Key": API, "Content-Type": "application/json"}, body: JSON.stringify({ range_name: "INDONESIA" }) } ); const data = await r.json(); console.log(data.added, "nomor ditambahkan");
Endpoint

Semua endpoint tersedia

Proxy langsung ke iVAS. Butuh X-API-Key di header atau ?api_key= di URL.

POST/api/ivas/loginLogin ke akun iVAS sendiri
GET/api/ivas/statusStatus koneksi iVAS + WebSocket
GET/api/sms/liveAmbil SMS live dari cache WS
GET/api/sms/live/streamSSE push stream real-time
GET/api/sms/receivedHistori SMS per range + nomor
GET/api/rangesDaftar range aktif
GET/api/numbersNomor di range tertentu
GET/api/numbers/myMy Numbers di akun iVAS
GET/api/check-numberCek ketersediaan nomor
POST/api/numbers/addTambah nomor (by ID / range / nomor)
POST/api/numbers/deleteHapus / kembalikan nomor
POST/api/ws/reconnectReconnect WebSocket iVAS
